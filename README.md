检查代理服务器是否有效

3.checkproxyipg文件是检查代理服务器是否能连上www.google.com，与上一个文件的方式一样，只不过这个是看能不能访问google.(判断方法有问题，仅仅判断返回中是否有google串，有些错误页面返回也有google串。)
